Video Library

A free Mac app for browsing huge local video folders, seeing thumbnails, tagging clips, saving prompts, exporting CSVs, and optionally using AI to organize a messy library.

Download for Mac Try browser demo View release

Requires macOS 13 or newer. The browser demo uses sample records and does not install anything.

What it does

Browse big folders Import local video folders, search them, and page through thumbnail grids without loading everything at once.
Save metadata Add tags, origin, notes, favorites, ratings, and reusable prompts, then export JSON backups or CSV spreadsheets.
Optional AI help Use local Ollama or your own connected workflow to suggest tags, names, prompts, and review queues.

Install

  1. Download and unzip Video Library.zip.
  2. Drag Video Library.app into Applications.
  3. Right-click the app and choose Open the first time macOS asks.
  4. If macOS still blocks it, use the included Sentinel helper or download from the release page.

Demo

The browser demo lets people click around the library, filters, tags, prompts, and metadata tools with fake sample videos. Real folder import, native Finder actions, local video preview, and Ollama setup are Mac app features.
